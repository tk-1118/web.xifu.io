    《第一次正式的约会》

    # 哈哈哈要和老婆大人见面啦，第一站约在19号线的牛街站带老婆去吃早餐

    小调皮的老婆大人还不告诉我他在哪个车厢，想给我一个惊喜哈哈哈（我转身就躲在了柱子后边，等她出来找不到我，我绕过柱子从背后抱紧她）

    # 来牛街排队了，搭在忆文的肩上记录老婆大人绝美的背影嘿嘿嘿

    # 带老婆大人来喝羊杂汤了咳咳~咳，哈哈

    老婆大人：我要告诉公主，我老公带我出来半个小时了还没吃上早饭哈哈哈，老公：咳咳~咳，失误失误哈哈哈，咱在这就去喝羊杂哈哈哈。（路上耗的太久，买的早餐有些凉了。老公是在不断进步的哈哈，第一次正式约会没有什么经验造成了失误，老公保证下次不会这样了）

    # 在老公怀里熟睡的大宝贝儿

    辛苦了我可爱的老婆大人了，为了见面五点半就起床了（让老公好好抱抱，睡吧~睡吧~有老公在呢）

    # 和老婆大人在朝阳大悦城挑选了一身夏季情侣装

    老婆大人的换装视频就放到我的收藏里，老公自己欣赏啦哈哈哈，恋爱网站就不挂啦

    （从一束花里面选出来的两朵红玫瑰，这么有意义的一次必须做成干花纪念一下）

    # 老婆大人给小老公买的面膜，被老婆宠着的感觉好幸福呀！哈哈哈

    # 下午三点啦，最后的二人小世界来到啦哈哈哈

    和老婆大人一起定了《隐阁·影格星顶影棚》---三小时双人观影+谷饲牛排+焦盐罗氏虾，哈哈哈这是只有我们两个人的私密小房间

    在最后的约会地点偷偷准备了一束老婆大人喜欢的红玫瑰（我的大宝贝呀！老公答应以后每次见面一定准备一束花怎么会忘记呢）

    老婆大人呀，恋爱一生一次就够了，我要牵着宝贝儿老婆走进婚姻的殿堂！！！

    # 打车把老婆送到了草桥地铁站，哎！看着大宝贝儿搭上列车回家的背影，老公是真的！真的！真的！爱我的老婆大人呀！呜呜~呜不舍，期待七月份的下次见面。
