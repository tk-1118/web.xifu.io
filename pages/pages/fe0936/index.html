七月约会之《环球影城》❤️💕😍

# 与老婆在双井地铁站见面啦嘿嘿

提前在双井站预约了外卖（蒸饺、灌汤包......），不能让大宝贝跟老公出去游玩饿肚子哈哈哈哈

七月三人小队正式集结完毕✌

# 到环球影城，门口的队伍真是“山里十八弯”好多人呀哈哈哈

还好有遮阳伞，不然就真晒成非洲难民了😱

# 历经漫长的排队，终于进来啦哈哈哈哈

# 闹了个大乌龙，公主的身份证丢了，只能拿着身份证截图与户口本截图去领临时通行卡，结果领完以后才发现身份证在包里哈哈哈哈哈

哈哈哈哈就当是环球影城的纪念品吧

# 历经“曲折”终于进来啦哈哈，第一站 飞跃侏罗纪✌

不愧是T0的热门项目之一哈哈，太刺激啦😘，被迅猛龙追赶，还有场景的布置真的就像身临其境，回到了侏罗纪！

# 第二站 哈利波特-环球影城顶流✌

在哈利波特的世界怎么能不来来一杯黄油啤酒捏~~~~

浅浅合照一番哈哈😘

不敢想像刚才还是晴空万里，不一会就乌云密布了~~~ 突然的清凉~不用打伞也哈哈哈

来《三把扫帚》吃饭啦, 真的很还原电影哈哈哈，太喜欢啦

# 第三站 水上世界😁

来的有点晚了，没抢到中心位置哇呜~~~~，坐在边缘的湿身区就像老年活动中心，不过剧情确实挺好看哈哈哈哈，体验感7分

# 第四站 功夫熊猫~阿宝~

唯一的纯室内游玩区哈哈，不用怕晒黑了

室内灯光不错，但可惜游玩项目偏向于儿童, 索性就多拍照啦哈哈哈哈

# 第五站原本决定去玩霸天虎呢，结果看到那个高度和104KM/H的速度还是望而止步了哈哈，保命要紧，我还想多陪陪大宝贝嘞

# 后面又看了小黄人和环球大巡游嘿嘿

# 重点重点来啦哈哈，当天旅行最喜欢的一对合照😍😍😍

# 哪个男生能这么幸福哈哈，我的宝贝老婆在给我系鞋带哎，心里有些小局促小紧张哈哈，但更多的是感觉很幸福！！！😊

# 旅行的结束, 我的宝贝老婆也笑得快岔气了哈哈哈哈哈哈~~~，环球之旅真的可以说是当下最安逸最快乐的约会😊

# （这次旅行又流鼻血啦！！！哇呜~~~被老婆和公主无情的“嘲笑”哈哈哈哈，好羞涩啊😳😳😳）
